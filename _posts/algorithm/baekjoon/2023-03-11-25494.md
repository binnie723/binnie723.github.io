[25494번: 단순한 문제 (Small)](https://www.acmicpc.net/problem/25494)


정말 단순한 문제였다. 완전 탐색으로도 해결되는 문제이지만, min을 사용하면 $O(n)$으로 풀 수 있다. 

<br/>
### 맞은 코드

```python
t = int(input())
for _ in range(t):
  a, b, c = map(int, input().split())
  print(min(a, b, c))
```
